# AkaDakoロガー

AkaDako（赤だこ）センサーボードのデータをブラウザ上でリアルタイムに記録・表示するWebアプリです。

## 機能

- AkaDakoセンサーボードとBluetooth接続
- 複数センサーの同時記録（明るさ、温度、湿度、気圧、距離、水温、加速度、ピッチ、ロール、音量、動き）
- 計測周期の選択（1秒〜60分）
- センサー列の追加・削除
- CSV形式でのダウンロード
- TSV形式でのクリップボードコピー（スプレッドシートに貼り付け可能）

## 使い方

1. `index.html` をブラウザで開く
2. 「AkaDakoに接続」ボタンでセンサーボードに接続
3. プルダウンから記録したいセンサーを選択
4. 「計測開始」ボタンで記録を開始

## 動作環境

- Web Bluetooth APIに対応したブラウザ（Chrome、Edgeなど）
- AkaDakoセンサーボード
